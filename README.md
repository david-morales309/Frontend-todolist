## Descripción

Proyecto sencillo de lista de tareas donde puedes:

- Añadir nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas
- (Opcional) Editar y filtrar tareas
- (Opcional) Persistencia en localStorage

> Si alguna de estas funcionalidades no coincide con tu código, puedo adaptar el README si me indicas los detalles.

## Instalación

1. Clona el repositorio:

   git clone https://github.com/savella11/Frontend-todolist.git

2. Entra al directorio:

   cd Frontend-todolist

3. Abre el archivo `index.html` en tu navegador o usa una extensión como Live Server en VS Code para servirlo localmente.

## Uso

- Escribe una tarea en el campo de entrada y pulsa Enter o el botón de añadir.
- Marca la tarea como completada para tacharla.
- Elimina tareas con el botón de basura.
